- grep  搜索文档内容 关键字  语法：grep  [参数]  关键字  文档   ；--- 显示包含关键字的哪一行信息
    + -n  显示行号   grep -n "AAA" 2.log
    + -i  不区分大小写
    + 匹配 [值或者范围]
    + . 表示一位 相当于  mysql中  _  下划线
    + .* 表示任意位数  相当于  %
    + m\{n\}：表示m连续重复n次
    + 固定行首^  和行末$  例子：grep -ni "^1.*4$"  1.log
    + 精确匹配，匹配单词的  "\<\>"  例子：grep -ni "\<43\>"  1.log
    + 非转义 使用\    grep -ni "3\.14" 1.log    ;  grep -ni '3\.\*14' 1.log  
    + 管道符 |  ， 前面的输出作为后边的过滤对象，前面计算的结果先不忙显示，再进行后边的二次过滤  grep -ni "^1" 1.log | grep -n 27 | grep -n 14
    + 以后再实际中使用 ps -ef | grep XXXX -- 搜索进程的
- sort 排序 
    + -n 数字排序
    + -r  倒序
    + -k n  对第几列的排序
    + sort -nr -k 3 XXX.log  对文档中第3列数字排倒序
- wc 统计     wc 文档
- cp 复制   语法 cp [-r] 源文件  目标文件
    + 如果是复制目录  需要加 -r 
        + 例子：cp -r  test2/ .. 
    + -s  参数是建立软连接的  在 ln章节再讲
- mv 略
- rm  略
- find 查找文件或者文档的
    + -name  以文件名查找 支持模糊搜索 
        + find / -name "*.log"  查找/目录下的以.log结尾的文件或者目录
    + -user 按照文件的属主查询；无用户属性
        + find  /  -user  haohan
    + -group 按照文件的属组查询；无组属性的
        + find  /  -group  haohan
    + -type  按照类型查找 d 目录  f 普通文件  l  链接
    + 可执行操作 -exec 命令 {参数} \;    例子：find -name "*.log" -exec rm -rf {} \;
## 文件管理
- 查看文档内容
    + cat  语法：cat 要查看的文件名；还可以针对多个进行合并查看
        + 一次性将文件内容全部显示到屏幕上
        + 一般用于查看小文件
        + -n 和 -b参数 的区别  对空行是否进行编号
    + head
        + head -n 文件   显示文件的头n行；如果不限定具体行数默认显示头10行
    + tail
        +  tail -n 文件   显示文件的尾n行；如果不限定具体行数默认显示尾10行
        + tail -f 日志文件 查看动态日志
        + tail -500f 日志 
    + more -- 分页显示
    + less 等同于 more 也是实现分页的   按 回车  空格  退出使用q键
    + vi 略
    + sed  固定语法 sed -n "/关键字1/,/关键字2/p" 文件
    + 重定向 > 覆盖   >> 追加
- 权限管理
    + linux有3种权限  r  w  x  读 写 执行
    + chmod -R ug+rwx 1.log
    + chmod -R 755 1.log
    + chgrp  改文件的组属性 语法：chgrp [-R] 新组名/组id 文件
    + chown 改文件的拥有者属性 语法1：chown [-R] 新用户名/用户id 文件 ；语法1：chown [-R] 用户名/用户id:新用户名/用户id 文件 
- 比较命令
    + diff
    + cmp
    + comm
    + 实际工作中一般使用工具进行代码比对 beyond-compare；npp++
- 查看文件类型  file 文件
- ls 
    + ll = ls -l
    + -l  显示详情
    + -a 显示所有的文件包含隐藏的文件
    + -lrt 按照修改/创建的时间排序倒序
    + -i 查看i节点
- ln  链接
    + 硬链接
        + 语法：ln 源文件  目的文件
        + 只能链接文件不能链接目录
        + 如果对其中一个文件进行修改，另外的也会同步修改
        + 硬链接不会增加i节点
        + 不能对不存在的文件进行链接
    + 软连接（符号链接）--- 快捷方式
        + 语法：ln  -s  源文件  目的文件
        + 使用绝对路径，否则有可能会出现无效的连接
        + 可以对文件和目录进行链接
        + 可以对不存在的文件进行链接
        + 链接后的链接文件会生成新的i节点
        + 可以跨系统
        + 源文件删除链接还是可以存在，只是无效的
